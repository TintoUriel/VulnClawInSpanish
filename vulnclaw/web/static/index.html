Cómo acceder a la interfaz completa

Compilar el frontend de React

Ejecuta en el directorio del proyecto:

cd frontend

npm install

npm run build

Luego vuelve a abrir la dirección proporcionada por vulnclaw web.

Flujo principal

Recorrido del usuario

  1. 1Ingresa una URL, dominio o IP explícitamente autorizada.
  2. 2Elige reconocimiento rápido, verificación estándar o verificación profunda.
  3. 3Confirma los límites de puerto, host, ruta y acción.
  4. 4Revisa los resultados de riesgo y exporta un informe en Markdown / HTML.
Límites de seguridad

No excedas el alcance autorizado

La interfaz web completa de VulnClaw muestra las restricciones only_*, blocked_* y de acción como límites legibles, y accede a la página de límites de seguridad para ver el motivo del bloqueo si se produce un intento fuera de alcance.

Se recomienda que las autopruebas comunes especifiquen al menos un host, puerto o ruta.

Si ya compilaste el frontend pero sigues viendo esta página, confirma que frontend/dist/index.html existe y reinicia el servicio web.
